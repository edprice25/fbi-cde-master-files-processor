The Cargo Theft Master File contains data that are submitted by each participating agency reporting to the UCR Program.
For each incident, the file provides
the month in which the incident occurred,
the date of the incident,
the hour the incident occurred or the report date indicator,
the cleared exceptionally indicator,
the offense(s),
location,
weapon/force involved,
the victim type,
property loss type,
date,
and value of property stolen and property recovered,
and the offender and arrest data by age, gender, race, and ethnicity.
